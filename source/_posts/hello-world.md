> 熟悉一下 Hexo 基本命令

### Create new post

``` bash
$ hexo new "My New Post"
```

### Run server

``` bash
$ hexo server   --简写-->   hexo s    
```

### Generate static files

``` bash
$ hexo generate    --简写-->   hexo g
```

### Deploy to remote sites

``` bash
$ hexo deploy    --简写-->   hexo d
```
